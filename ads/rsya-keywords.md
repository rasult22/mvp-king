# Ключевые слова для РСЯ (Рекламная сеть Яндекса)

Продукт: разработка MVP под ключ за $2000 / 2–3 недели, один человек.
Аудитория: фаундеры/предприниматели с идеей, которую хотят быстро проверить.

> В РСЯ фразы работают широко (показ тем, кто искал/читал тематику).
> Берём средне-широкие фразы по теме, мусор режем минус-словами и по статистике.
> Заводим ОТДЕЛЬНЫЕ группы объявлений под каждую смысловую группу.

## Нюансы запуска

1. Отдельные группы объявлений под Гр.1–4 (разный интент → раздельная статистика).
2. Группа 2 — самая точная, начать бюджет с неё.
3. Включить автотаргетинг параллельно с ключами.
4. Определиться с гео/языком (РФ / СНГ / русскоязычные за рубежом) — влияет на ставки и список.
5. Перед запуском: Яндекс.Метрика + цель на отправку формы заявки (без этого РСЯ не оптимизируется).
