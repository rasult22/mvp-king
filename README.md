## Запуск

Статический одностраничник без сборки — открыть `index.html` в браузере.

## Стек

Чистый HTML / CSS / JS, без зависимостей и фреймворков. Шрифт — Space Grotesk (Google Fonts).

## Что внутри

- Два полноэкранных блока с fullPage-пейджингом (один скролл = переезд на экран).
- Кинетическая типографика, тёмная сцена, акцент `#c6ff3a`.
- Живость на курсоре: спотлайт-свет + параллакс строк (без 3D/WebGL).
- CTA «Связаться» → модалка: Telegram или форма заявки.
